# 중단 - 업무 도구
- PPT를 PDF로 변환
- PDF를 PNG로 변환
- IMG(PNG, JPG)를 PDF로 변환
- IMG 회전
- PDF 합치기
- PDF 분할 및 페이지 삭제
- PDF 용량 조절(압축)
- PDF 암호 설정
- PDF 암호 해제
- PDF 비교

- 제안서 패키징
    1. 패키징 할 PPT 업로드
    2. 업로드한 PPT의 순서 조정
    3. 준비된 PPT를 PDF로 변환
    4. 변환된 PDF를 1개 파일로 통합
    5. 통합된 PDF를 장별로 순서 조정
    6. 필요 시 용량 조절(압축)
    7. 최종 결과 PDF 저장
