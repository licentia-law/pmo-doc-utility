# 1. 제품 개요

## 1.1 제품명 (가칭)

* PMO Portal
* DocMate for Proposal
* Proposal Utility Suite

## 1.2 제품 목적

사내 제안서 작성 및 제출 과정에서 반복되는 문서 작업과 사이트 접근을
**하나의 데스크톱 앱에서 빠르게 처리**할 수 있도록 한다.

---

# 2. 제품 한 줄 정의

👉
**자주 사용하는 업무 사이트 + 문서 작업을 한 화면에서 빠르게 처리하는 사이드형 업무 런처**

---

# 3. 제품 목표 및 핵심 가치

## 3.1 핵심 가치

👉 **단순함 (가장 중요)**

* 누구나 바로 사용 가능해야 한다
* 복잡한 옵션은 최소화한다
* “버튼 클릭 → 결과” 구조

---

## 3.2 목표

* 문서 작업 시간을 단축한다
* 반복 작업을 줄인다
* 여러 툴을 오가는 흐름을 하나로 통합한다
* 작업 실수를 줄인다

---

# 4. 제품 사용 방식

## 4.1 사용 형태

👉 **항상 켜놓는 사이드형 보조 도구**

* 메인 작업: PPT / 브라우저
* 본 앱: 실행/보조 역할

---

## 4.2 창 정책

* 화면 한쪽에 고정 사용
* 최대 폭: 모니터의 50% 이하
* 세로형 패널 구조

---

# 5. 핵심 기능 범위 (MVP 확정)

## 5.1 홈 화면 노출 기능 (6개)

4. PDF 용량 줄이기
5. 제안서 패키징
6. 업무 매뉴얼

---

## 5.2 보조 기능 (홈 미노출)

* 작업 기록
* 설정
* About
* 업데이트

# 7. 사용자 경험(UX) 원칙

## 7.1 핵심 원칙

👉 **생각 없이 사용할 수 있어야 한다**

* 클릭 중심 UX
* 옵션 최소화
* 빠른 실행

---

## 7.2 인터랙션 원칙

* 입력 단계 최소화
* 설정은 숨김
* 기본값은 안전한 값

---

## 7.3 실패 UX

* 간단한 메시지만 제공

  * “실패했습니다”
  * 이유 1줄

❌ 해결 방법 설명 없음
❌ 기술적 상세 설명 없음

---

# 8. 기능 상세 정의

## 8.1 PPTX → PDF

* 단일 파일 변환
* 드래그 앤 드롭 지원
* 저장 경로 선택
* 결과 폴더 열기 가능

---

## 8.2 PDF 병합

* 여러 PDF 선택
* 순서 변경 가능
* 하나의 PDF로 병합

---

## 8.3 PDF 용량 줄이기

옵션:

* 최고품질
* 균형
* 소용량
* 목표 용량 입력

---

## 8.4 제안서 패키징

* PPTX + PDF 혼합 입력
* 순서 정렬
* PPT 자동 PDF 변환
* 전체 병합
* 선택적 용량 조절

---
