# PMO Doc Utility

사내 제안서 작성 및 제출 업무를 위한 **문서 처리 + 업무 런처 통합 데스크톱 앱**

---

## 📌 프로젝트 소개

PMO Doc Utility는 제안서 작업 과정에서 반복적으로 발생하는 문서 처리 업무를 빠르게 수행하기 위한 **Windows 기반 데스크톱 유틸리티**입니다.

PPTX → PDF 변환, PDF 병합, 용량 조절, 제안서 패키징을 하나의 흐름으로 처리하며,  
자주 사용하는 업무 사이트와 체크리스트까지 한 화면에서 제공합니다.

---

## 🎯 주요 목적

- 반복적인 문서 작업 자동화
- 제안서 제출 준비 시간 단축
- 문서 품질 유지 (PowerPoint 기반 변환)
- 여러 도구/사이트 이동 없이 한 곳에서 업무 처리
- 오프라인 환경에서도 동작 가능한 로컬 처리

---

## 🚀 핵심 기능

### 1. PPTX → PDF 변환
- PowerPoint 기반 고품질 변환
- 로컬 환경에서 처리 (보안 안전)
- 폰트/레이아웃 유지 우선

---

### 2. PDF 병합
- 여러 PDF 파일을 하나로 병합
- 드래그로 순서 변경
- 제출용 문서 구성 최적화

---

### 3. PDF 용량 조절
- 제출 제한(MB 기준)에 맞춰 압축
- 옵션:
  - 최고품질
  - 균형
  - 소용량
  - 목표 용량 지정

---

### 4. 제안서 패키징 (핵심 기능)
- PPTX + PDF 혼합 입력 지원
- 자동 변환 + 병합 + 압축까지 일괄 처리
- 최종 제출 파일 자동 생성

---

### 5. 업무 사이트 즐겨찾기
- 자주 사용하는 사이트 즉시 접근
- 기본 브라우저로 실행

---

### 6. 업무 매뉴얼 / 체크리스트
- 제출 전 확인사항 제공
- 자주 발생하는 실수 방지
- 간결한 카드/아코디언 형태

---

### 7. 작업 기록 관리
- 수행 이력 확인
- 개별/전체 삭제 지원
- 로컬 저장 (외부 전송 없음)

---

### 8. 라이트 / 다크 모드
- 사용자 환경에 맞는 테마 지원
- 로컬 설정 저장

---
